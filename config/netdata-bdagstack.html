bdagStack metrics

Chain / RPC + mining charts come from Netdata job bdag_chain_pool_local (exporter 127.0.0.1:9198/metrics). Node native charts come from bdag_node_native_local (:6060/metrics). This page uses the Netdata /api/v1/data API (no legacy dashboard.js), so it works on Agent v2/v3 where the classic web bundle is not installed. Open from the agent, e.g. http://127.0.0.1:19999/bdagstack.html.
